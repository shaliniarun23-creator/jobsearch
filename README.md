# Job Hunt Live Job Finder & Ranking Dashboard

This Streamlit app finds live job postings from trusted job sites and ranks them based on relevance, fitment, location, experience level, and live-job status.

## Features

- Finds jobs from trusted platforms:
  - LinkedIn Jobs
  - Indeed
  - Naukri
  - Foundit
  - IIMJobs
  - Wellfound
  - Greenhouse
  - Lever
  - Workday

- Scores jobs based on:
  - Relevance to profile
  - Role fitment
  - Location match
  - Years of experience
  - Live/open job signals
  - Rejection filters

- Produces:
  - Ranked job table
  - Top jobs to apply first
  - Downloadable CSV file
